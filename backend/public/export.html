Exportar Peças (Exemplo)

Cole aqui o token JWT retornado pelo login (ou execute o login via console) e faça o download do CSV.

Dica: abra o DevTools → Console e execute:
localStorage.setItem('token', 'SEU_TOKEN_AQUI') para gravar o token usado por padrão.

Exemplo rápido de login (browser console)

fetch('/api/login',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({email:'seu-email@exemplo.com',senha:'suaSenha'})})
  .then(r=>r.json()).then(j=>{ console.log(j); if(j.token) localStorage.setItem('token', j.token); })
